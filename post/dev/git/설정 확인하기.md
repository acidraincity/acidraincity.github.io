다음 명령을 사용해서 현재 적용되어있는 git 설정을 확인할 수 있습니다.

```bash
git config --list
```

그런데 같은 설정이 중복되어 표시되는 경우가 있을 수 있는데요.

설정파일이 스코프별로 여러개로 나뉘어 있을 수 있기 때문에 그렇습니다.

아래 명령으로 어떤 설정파일에 어떤 설정이 들어있는지 표시해볼 수 있습니다.

```bash
git config --list --show-origin
```
